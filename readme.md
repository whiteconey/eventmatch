## EventMatch

EventMatch — десктопное приложение, предназначенное для поиска мероприятий по заданным критериям и интересам. Программа позволяет пользователям находить актуальные события, а организаторам — эффективно управлять публикациями.

## Функциональные возможности

**Поиск и фильтрация:** Реализован поиск мероприятий с фильтрацией по категориям, датам, локации и тегам.
**Профили пользователей:** Интегрирована функция сохранения выбранных событий в раздел «Избранное».
**Панель организатора:** Доступны инструменты создания, редактирования и удаления карточек мероприятий.
**Хранение данных:** Обеспечена синхронизация с реляционной базой данных для быстрого доступа к информации.

## Инструкция по установке и запуску

### Установка исполняемого файла (для пользователей)
1. Перейдите в раздел **Install_eventmatch** текущего репозитория.
2. Скачайте файл установщика setup.exe или EventMatch_Setup.msi.
3. Запустите загруженный файл и следуйте инструкциям мастера установки.
4. Запустите приложение с помощью MyApp_EventMatch.exe в папке, куда была осуществлена установка.
